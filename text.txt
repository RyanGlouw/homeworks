Команды Git

git rm -r --cached "путь" удалить из индекса

git commit -m "" сохр коммита

если локальные изменени не нужны то - спрятать локальные изменения git stash - скрытие (прятание)

или git restor имя файла - удаляет локальные изменения
git restor. - удаляет все локальные изменения

если локальные изменени нужны то - делаем git add и далее git commit и открываем конфликтный файл
